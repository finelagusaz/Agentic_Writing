# 🖊️ Agentic Writing

**AIマルチエージェントによるWeb小説自動執筆プラットフォーム**

---

## 概要

Claude Code のマルチエージェント機能を活用し、編集者・作者・担当者・読者の4役割を持つAIチームがエピソードを自動執筆します。ジャンルに依存しない汎用フレームワークで、対話的セットアップにより任意の作品を書き始められます。

## 特徴

- 🎭 **マルチエージェント**: 編集者が方針を立て、作者が執筆し、担当者がレビューし、読者がフィードバックする
- 🎯 **ジャンル非依存**: ライトノベル・ミステリー・青春小説・SF など何でも対応
- 🇯🇵 **日本市場特化**: なろう系、ラブコメ、BL等の日本固有ジャンル分類とアーキタイプ
- 🤖 **AI癖制御**: Show, don't tell、文末バリエーション、独自用語禁止などのAI文体制御を内蔵
- 🔄 **自動リビジョン**: 品質が基準に満たない場合、自動で改稿サイクルを回す

## クイックスタート

```bash
# 1. 作品のセットアップ（対話的7ステップ）
/setup-story

# 2. 資料の修正（任意）
/edit-story 主人公の年齢を20歳にしたい

# 3. エピソード執筆
/write-episode 1
```

## スキル一覧

| スキル | 説明 |
|--------|------|
| `/setup-story` | ジャンル選択→テーマ→世界観→キャラ→プロット→文体→読者ペルソナの7ステップで作品を構築 |
| `/edit-story` | 自然言語で資料を修正。波及影響を自動分析 |
| `/write-episode N` | 第N話を自動執筆。レビュー・フィードバック・リビジョンを含む |

## エージェント構成

```
┌─────────────────────────────────────┐
│            リーダー（あなた）           │
│         オーケストレーション            │
├─────────┬─────────┬─────────────────┤
│  編集者   │  作者    │    担当者        │
│ (opus)   │ (opus)  │   (sonnet)      │
│ 方針策定  │ 執筆    │   品質レビュー    │
├─────────┴─────────┴─────────────────┤
│     読者ペルソナ1・2・3 (sonnet)       │
│     バックグラウンドでフィードバック      │
└─────────────────────────────────────┘
```

## 執筆ワークフロー

```
方針策定 → 議論 → 執筆 → レビュー → 議論 → 読者FB → 判定
    │                                                │
    │  ← ← ← ← ← リビジョン（最大3回） ← ← ← ← ← ←  │
    │                                                │
    └ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ 確定 ─ ─ ─ ─ ─ ─ ─ ─ ─ ─┘
```

## 前提条件

- [Claude Code](https://docs.anthropic.com/en/docs/claude-code) のサブエージェント機能が利用可能なこと
- Claude Code の `opus` モデルと `sonnet` モデルが利用可能なこと
